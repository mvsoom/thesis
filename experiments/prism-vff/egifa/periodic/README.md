# prism-vff/egifa/periodic

Smoke test of PRISM-VFF on EGIFA test data

A remake of experiments/lvm/egifa/{periodic,pack} BUT with

- x32: stable and very fast
- 16 iterations with random inits ~ LogNormal(0,1)
- test set evaluation with log evidence of the implied PRISM BLR
- with harmonics null model: same variance and inducing inputs, but approximately constant spectrum
